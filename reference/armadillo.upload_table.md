# Uploads a table to a folder in a project

Uploads a table to a folder in a project

## Usage

``` r
armadillo.upload_table(project, folder, table, name = NULL)
```

## Arguments

- project:

  the project to upload to

- folder:

  the folder to upload to

- table:

  the table to upload

- name:

  name of the table (optional)

## Examples

``` r
if (FALSE) { # \dontrun{
armadillo.upload_table(
  project = "gecko",
  folder = "core_all",
  table1
)
} # }
```
