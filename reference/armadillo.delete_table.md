# Delete table

Delete table

## Usage

``` r
armadillo.delete_table(project, folder, name)
```

## Arguments

- project:

  project to delete the table from

- folder:

  folder to delete the table from

- name:

  table name

## Examples

``` r
if (FALSE) { # \dontrun{
armadillo.delete_table(
  project = "gecko",
  folder = "core_all",
  name = "table1"
)
} # }
```
