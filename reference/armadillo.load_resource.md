# Load a resource from a project

Load a resource from a project

## Usage

``` r
armadillo.load_resource(project, folder, name)
```

## Arguments

- project:

  study or collection variables

- folder:

  the folder containing the resource

- name:

  name of the resource

## Value

the loaded resource

## Examples

``` r
if (FALSE) { # \dontrun{
armadillo.load_resource(
  project = "gecko",
  folder = "core_all",
  name = "lc_core_1"
)
} # }
```
