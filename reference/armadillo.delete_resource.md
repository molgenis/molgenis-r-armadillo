# Delete resource

Delete resource

## Usage

``` r
armadillo.delete_resource(project, folder, name)
```

## Arguments

- project:

  project to delete the resource from

- folder:

  folder to delete the resource from

- name:

  resource name

## Examples

``` r
if (FALSE) { # \dontrun{
armadillo.delete_resource(
  project = "gecko",
  folder = "core_all",
  name = "table1"
)
} # }
```
